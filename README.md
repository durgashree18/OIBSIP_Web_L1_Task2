# Personal Portfolio Website

This project is a simple and responsive **Personal Portfolio Website** created using **HTML and CSS**. The website is designed to showcase personal information, projects, resume, and contact details in a clean and organized layout.

## Project Overview

The portfolio website contains different sections that introduce the developer and display their work. It includes a navigation bar for easy access to sections like About, Projects, Resume, and Contact.

The layout is styled using CSS and includes responsive design to ensure the website works well on smaller screens.

## Features

- Navigation bar for easy page navigation
- Header section with profile image and introduction
- About section describing the developer
- Projects section showcasing sample projects
- Resume section with downloadable resume link
- Contact section with email, LinkedIn, and GitHub links
- Footer section with copyright information
- Responsive design for mobile devices

## Technologies Used

- HTML5
- CSS3

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html**.
4. The portfolio website will open in your web browser.
